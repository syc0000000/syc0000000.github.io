hexo 显示图片的解决方案

  • 方案一

    使用 images 文件夹(适合链接网站图标). 在 sourse/images 文件夹中放入图片, 然后在博客文档里面地址写: /images/xxxx 或者其他任何需要引用图片的地方, 例如

  • 方案二 ( 推荐 )

    使用 hexo 官方的方案, 在配置文件中写入: ( 不需要安装插件 )

    然后每次在创建新文件时, 会自动创建一个同名文件夹, 然后引用图片时写: 1.png ,然后同名文件夹中的1.png就会被引用( 注意区分大小写 ) . 推荐用法: 在写文档时吧文件和文档放在同一个文件夹, 可以实时预览, 在写作完成后, 直接全部拖入同名文件夹中上传.
